# Traffic Violation & Penalty System

## Project Objective
To manage traffic violations, calculate fines, track repeat offenders, and generate a summary report with revenue collection.

## Development Steps
1. Designed a `Record` structure to store license, violation type, fine, payment status, and repeat count.
2. Implemented functions to:
   - Insert new violation records.
   - Update penalties for repeat offenders.
   - Mark violations as paid.
   - Display summary report.
   - Save data to disk.
3. Created a menu-driven interface in C.
4. Used file I/O to save records.

## How to Use
1. Compile the code: `gcc traffic.c -o traffic`
2. Run the executable: `./traffic.exe`
3. Follow the menu options:
   - Add violation
   - Apply penalties
   - Mark as paid
   - Show summary
   - Save & exit
